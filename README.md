# LTcoaching ( Version 1 ) 🏋️‍♂️

> Plateforme moderne de coaching sportif et bien-être.

LTcoaching est un site conçu pour présenter des services de coaching personnalisé, des programmes d'entraînement et faciliter la prise de contact entre le coach et ses clients.

# Installation

Pour explorer le projet localement :

1. / Cloner le dépôt /

   ```bash
   - git clone [https://github.com/ThibautTobi/LTcoaching.git](https://github.com/ThibautTobi/LTcoaching.git)

   - cd Projet_LTcoaching_Shadcn\lt-coaching

   - npm install

   - npm run dev

   - http://localhost:3000
   ```
